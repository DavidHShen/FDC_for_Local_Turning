# Factorized Derivative Calculus for Local Turning (FDC-LT)

This repository contains a GitHub-ready single-file implementation script.

Run:

```bash
pip install -r requirements.txt
python FDC_for_Local_Turning.py
python FDC_for_Local_Turning.py --output-dir outputs/fdc_lt --dtype float64 --seed 20260522
```

The script writes CSV diagnostics to `results/` and PNG figures to `figures/` under the selected output directory.
